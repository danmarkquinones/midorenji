FIGURA Visit Website

E-commerce Website Target : Action Figure Collectors Languages: HTML 5 | CSS | Bootstrap | Javascript | jQuery | PHP | MySQL

About the Page: FIGURA is an e-commerce website, a shop designed for action figure collectors, here you can find fully articulated Gunpla. If you like MCU we also have Hot Toys Avengers: Infinity War Figures. We also have Anime Figures for all Otaku out there. All items here are brand new and made from high quality materials.

This project enhanced my logical skills using Javascript/jQuery and PHP connected to a MySQL database. This websites includes the following functionalities

  • Account - A user can register and log-in.
  • Shop - A user can shop and add it to their cart.
  • Payment Method - After shopping the user can choose either COD or Paypal.
  • Mailer - Once the user bought an item, they will recieve an email coming from the website.
  • Admin Panel - A user has a privilage to be an admin. By that, they can add/edit/delete some items in the websites. An admin is also responsible on orders status.
  • Chat - The buyer can contact the admin using Messenger Chat connected to the facebook page of FIGURA.
